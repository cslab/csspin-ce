spin_ce documentation
=====================

The ``spin_ce`` plugin-package for cs.spin serves to provide the necessary
plugins and tools for building and testing CONTACT Elements instances, as well
as starting and managing its services.

The sources can be found at https://code.contact.de/qs/spin/spin_ce, while
the released versions are available at
https://packages.contact.de/tools/misc/spin-ce.

``spin_ce`` requires at least Python 3.9.
